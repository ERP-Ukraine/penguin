wallee extension from PIT Solutions is a connector between wallee and Odoo Web Shop.

wallee is a standardized way to accept payments via a multitude of different payment providers in the world. It is a single payment gateway to send payments to Adyen, Braintree, Ingenico, Heidelpay, PostFinance, Concardis, FirstData, Datatrans, SagePay, Barclaycard, etc. Beside the payment processing wallee offers a wide range of additional features like reconciliation. More information can be found under wallee

Our extension connects Odoo shop user to the payment hub, which has more than 50 payment gateways and processors linked to it. The Connector works with the help of the Rest API's. This reduces the tendency of customers abandoning their cart when their preferred payment gateway is missing in the payment page. It also connects the user to the different payment service provider APIs and thereby lowering the costs of using payment routing.

Please Note

All payment methods available in wallee are supported in our odoo extension.

wallee back-end provides a configuration wherein one can define the Payment methods to be enabled for users. In addition to this, admin has the flexibility to specify for the payment completion process the user should be redirected to the service’s provider’s site or handled within shop site as I-frame. It depends on the payment service provider whether the gateway can be handled via redirection or I-frame or both.

For an additional security level in the payment process, Admin can enable the 3D secure functionality in the wallee back-end. If enabled, then during the checkout process user would be directed to the 3D secure service of the payment method and the details would be handled there.

Admin can change the notification contents in the back-end, for cases when the payment is pending, completed, cancelled or an error occurs. Other system notifications are help message when the Hel2 option is used and Thanks message on completion of the order placement. Similarly, the order emails, invoices, reminders, packing slips and many more can be edited as per one’s format.

Compatibility

The extension is compatible with ODOO version 9, 10, 11, 12, 13, 14 and 15 and it’s easy to install. If installed in Version 9, it can be easily migrated to version 10 and 11 on need.

Please note that for some payment methods you are required to create a payment service providing contract with the payment service provider like Braintree, SagePay etc
