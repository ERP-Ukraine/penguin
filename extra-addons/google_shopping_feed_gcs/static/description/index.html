What is Google Shopping(Merchant Center)?

Google Merchant Center helps you get your shop and product info into Google and make it available to shoppers across Google. That means that everything about your shops and products is available to customers when they search on a Google property.
Google Shopping is a new product discovery experience. The goal is to make it easy for users to research purchases, find information about different products, their features and prices, and then connect with sellers to make their purchase.
Visit link for more information: https://www.google.com/retail/solutions/merchant-center/

Google Shopping/Merchant Account Create & Setup

First you have Google Merchant account. If you haven't created then visit following link and create your Google Merchant Account.
https://www.google.com/retail/get-started/?product=merchant-center
At the registration time ask Business Name, Business Country, Time Zone, ..etc so fill-up it and create your account in Google Shopping.

After, Creating Google Merchant account will look like,

Google Console Account Create & Setup

After, Created account in Google Merchant. Required to create and setup google console account for connection with Odoo.
If you have not account then following setup to you can create Google Console account. 

1. Goto https://console.cloud.google.com/
2. Login with your gmail account.
3. Goto Dashboard there is option for New Project so click on it.
	3.1 Project Name (i.e GCS Odoo API) you can insert anything.
	3.2 Location you can leave it.
	3.3 Click on Create button to create your project.
4. After need to Enable Google Shopping API service,
	4.1 Search "Content API" in search bar of the console screen.
	4.2 There is API name is "Content API for Shopping" so click on it.
	4.3 Click on Enable button to enable this API for google shopping.
5. After, Need to create API credential so that for goto "API & Service" > Credential Menu.
	5.1 On the header have "+ Create Credentials" button so click on it and select "OAuth Client ID" option.
	5.2 After, If you have not create and setup "Configure Constant Screen Record" then you will redirect the another screen where you can see "Configure Constant Screen" so click on it.
	5.3 Select "External" option then click on Create.
	5.3 After, fill-up all the information properly and click on Save and Continue button.
6. Left side you can see "Credential" menu so click on it.
	6.1 On top header "+ Create Credential" button so click on it.
	6.2 Select OAuth Click ID.
	6.3 Select application type "Web Application"
	6.4 Fill-up following information,
		>> Name: i.e GCS Odoo API
		>> Authorized JavaScript origins: Insert here your website URL (i.e http://www.growconsultancyservices.com)
		>> Authorized redirect URIs: Insert here redirect URI. Pattern of the URI should be like, http://FullDomainName/google_account/authentication2
			>> Example: http://www.growconsultancyservices.com/google_account/authentication2
		>> Click on Create button
	6.5 After you will get Client ID and Client Secret Key so save it.

Create & Configure Google Shopping Account in Odoo

Following steps to you can create and configure google shopping account in Odoo, 

1. Goto Google Shopping App > Configurations > Shopping Account Menu.
2. Create new account and fill-up all required information,
	* General Information Tab
	2.1 Shopping Account Name: i.e. Grow Consultancy Services
	2.2 Company: Select your company.
	2.3 Warehouse: Select warehouse for stock and order process.
	2.4 Stock Field: Product stock will export to Google Shopping from selected field here.
	
	* Google Shopping Setting
	** Website Information
	2.5 Website: Select website here. Product will export to Google Shopping and Customers will redirect from Google Shopping to your selected website when they go for purchase.
	2.6 Website Domain: Select website domain carefully. This domain should be match with "Authorized JavaScript origins" which entered in Google Console.
	2.7 Authorized Redirect URI: It will automatic generate from website domain field. This also should be match with "Authorized redirect URIs" which entered in Google Console.
	
	** Shopping API Credentials
	2.8 Merchant ID: You can get from Google Merchant account and insert Merchant ID here.
	2.9 Client ID: You can get from Google Console and insert here.
	2.10 Client Secret: You can get from Google Console and insert here.
	
	** Claim/Verify Website
	2.11 Verification HTML File Name: Following steps to you can get Google Verification HTML file,
		2.11.1 Goto Google Merchant account.
		2.11.2 Click on Gear Icon, Then, Goto Tools > Business Information Menu.
		2.11.3 Then, Goto Website tab
		2.11.4 After, You need to insert domain name under the "Your website address" then click on Continue button.
		2.11.5 After, Verify/Claim website for click on "Add an HTML tag or Upload an HTML file you your website"
		2.11.6 Then, Click on "Upload an HTML File"
		2.11.7 Then, copy only file name and insert in Odoo's field "Verification HTML File Name". i.e google99236f339597f852.html
	2.12 Claim/Verify Website Button: Click this button to confirm the upload file successfully.
		>> You can also verify via link which will show during 2.11.6 step.
		For example: http://www.growconsultancyservices.com/google99236f339597f852.html
	2.13 Again, goto step 2.11.5 on Google Merchant Account, There is button "Verify Your Website" click on it.
		>> You will receive a confirmation email from Google Search Console and be able to continue to the next step (claim your website).

After, Click on "Authorize Shopping Account" button you will redirect to google account link

Successfully authorize after, again goto Google Shopping Account screen in Odoo and see state is changed to "Authorized". After, Click on "Get Auth Token" button to you will get authorize token for connection with Google Merchant.

Create Country Wise Shopping Instance

After, Created successfully Shopping Account in Odoo. You have to create country wise shopping instance in Odoo.
Following steps to you can create/configure Shopping Instance, 

1. Goto Odoo Google Connector App.
2. Goto Configurations > Shopping Instance.
3. Create Instance and Fillup all required information.
4. Click on "Confirm Shopping Instance" button to confirm instance.

Prepare Products For Export in Google Shopping

Following steps to you can prepare products for export in google shopping, 

1. Goto Sales > Products > Product.
	1.1 Select single/multiple products and Open Action dropdown menu, There is option "Prepare Product For Export in Google Shopping".
	1.2 After open Popup so select instance, which instance for you want to prepare for export. Then, Click on Prepare button to selected products will prepare not final export in Google Shopping.
2. After, Goto Odoo Google Shopping App > Products > Product Variants Menu. There you can see all prepared products.
	2.1 Open product record and you can fillup all information which you would like to show in Google Shopping.
	Note: Price, Stock and Images will directly take from Sales > Products record.

Export Products In Google Shopping

Following steps to you can export products in google shopping, 

1. Goto Odoo Google Shopping Connector App.
2. Then, Goto Products > Product Variants Menu.
3. Select products which you want to export in Google Shopping.
	3.1 Click on Action dropdown there is option "Export Products in Google Shopping" so click on it.
	3.2 Then, Popup will open so click on Export button to product will export in Google Shopping.

After, Successfully products exported in Google Shopping, "Export in Google Shopping" and "Active in Google Shopping" will be mark as True in Odoo.

Products will looks like below in Google Merchant Center after exported in Google Shopping.
